学习C语言总结（一）


#注：简要记录我有点理不清的内容，可能不全面，或有错误，以后更改。

定义变量： type variable
定义常量：

  • #define identifier value
  • const type variable value
    #区别：const类似于定义变量，只不过只可读

存储类：

  • auto：默认的局部变量
  • register：寄存器中，大小很小
  • static：在作用域内一直保存
    1. 静态局部变量，函数中
    2. 静态全局变量，文件中，默认的全局变量
  • extern：全局变量，用于文件之间
    #static 和 extern 都放在静态存储区，auto 放在栈中

siezeof：返回变量大小
&：返回变量地址
*：指向一个变量

函数定义：
return_type function_name (parameter_list)
{
function_body
}

#无返回用 void
